# vue3-admin-template

## 项目拉取

```bash
# 克隆项目
git clone https://github.com/Ace627/vue3-admin-template.git

# 进入项目目录
cd vue3-admin-template

# 安装依赖 | 建议不要直接使用 cnpm 安装依赖，会有各种诡异的 bug
pnpm install

# 启动服务
pnpm dev
```

## 目录参考

```bash
|-src
| |-config
|   |-defaultSetting.ts               # 项目默认配置
|   |-white-list.ts                   # 路由免授权白名单
```
